\# StudentRecordSys



\## Overview



StudentRecordSys is a Java Swing desktop application developed to manage student records. The system provides a user-friendly graphical interface that allows users to add, view, update, delete, and search student information efficiently.



\## Technologies Used



\- Java

\- Java Swing

\- Maven

\- IntelliJ IDEA



\## Features



\- User login system

\- Add student records

\- View student information

\- Update student details

\- Delete student records

\- Search student records

\- Graphical user interface (GUI)

\- Structured object-oriented design



\## Project Structure



\## How To Run



1\. Clone this repository.

2\. Open the project in IntelliJ IDEA.

3\. Allow Maven dependencies to download.

4\. Run the `Main` class to start the application.
